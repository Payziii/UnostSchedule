# 📅 UnostSchedule

Телеграм-бот для просмотра расписания занятий по группам. Поддерживает inline-режим, рассылки и уведомления об обновлениях расписания.

---

## 🗂 Структура проекта

```
├── index.js                 # Точка входа — запуск бота и сервера
├── config.js                # Константы: API, часовой пояс, список админов
├── db.js                    # Работа с SQLite (пользователи, фильтры)
├── api.js                   # Запросы к API расписания
├── keyboards.js             # Inline-клавиатуры выбора курса и группы
├── broadcast.js             # Логика рассылки + хранение состояния
├── server.js                # Express-сервер, эндпоинт /internal/notify
├── groups.json              # Список курсов и групп
├── .env                     # Переменные окружения (не коммитить!)
└── handlers/
    ├── commands.js          # Команды: /start, /today, /tomorrow и др.
    ├── callbacks.js         # Обработка кнопок (callback_query)
    ├── messages.js          # Диалог настройки рассылки
    └── inline.js            # Inline-запросы (@bot ИМЯ_ГРУППЫ)
```

---

## ⚙️ Установка

```bash
git clone https://github.com/Payziii/UnostSchedule.git
cd UnostSchedule
npm install
```

Создайте файл `.env`:

```env
BOT_TOKEN=ваш_токен_от_botfather
ADMIN_IDS=123456789,987654321
```

Запуск:

```bash
node index.js
```

## 🤖 Команды бота

### Пользовательские

| Команда | Описание |
|---|---|
| `/start` | Выбрать курс и группу |
| `/restart` | Сбросить группу и выбрать заново |
| `/today` | Расписание на сегодня |
| `/tomorrow` | Расписание на завтра |
| `/week` | Расписание на неделю |
| `/week <url>` | Расписание на неделю с кастомной ссылкой на таблицу |
| `/query <текст>` | Поиск по преподавателю, кабинету или предмету |
| `/rasp` | Расписание звонков |

### Админские

| Команда | Описание |
|---|---|
| `/stats` | Статистика: кол-во юзеров, топ групп и курсов |
| `/search <id>` | Найти пользователя по Telegram ID |
| `/broadcast` | Запустить рассылку (всем / по курсу / по группе) |

## 🔍 Inline-режим

Введите в любом чате `@UnostScheduleBot ИМЯ_ГРУППЫ` (например `@UnostScheduleBot ИСП-1308`).

Бот предложит три кнопки: расписание на неделю, сегодня и завтра — без перехода в личку.

---

## 📢 Уведомления об обновлениях расписания

ISP3304 API может отправить POST-запросы с изменениями расписания на:

```
POST http://localhost:5000/internal/notify
```

```json
{
  "group": "ИСП-1308",
  "changedDays": ["ПОНЕДЕЛЬНИК", "СРЕДА"],
  "filePath": "/tmp/schedule_ISP-1308.png",
  "url": "https://docs.google.com/..."
}
```

Бот автоматически разошлёт фото всем подписчикам этой группы.

## 🗄 База данных

SQLite-файл `users.db`, таблица `users`:

| Поле | Тип | Описание |
|---|---|---|
| `user_id` | INTEGER | Telegram ID пользователя |
| `course` | TEXT | Выбранный курс |
| `group_name` | TEXT | Выбранная группа |

---

## 🌐 API расписания

Бот обращается к `ISP3304 API`

| Эндпоинт | Параметры | Описание |
|---|---|---|
| `/o/schedule` | `day`, `group`, `course` | Расписание на конкретный день |
| `/o/week` | `group`, `course`, `[url]` | Расписание на неделю |
| `/o/search` | `query` | Поиск по строке |
| `/o/rasp` | — | Расписание звонков |

Все эндпоинты возвращают `image/png` или JSON с `{ status: false }` при ошибке.

---

## 🔒 Права и доступ

- `ADMIN_IDS` — список Telegram ID через запятую в `.env`
- Команды `/stats`, `/search`, `/broadcast` доступны только администраторам
- Callback-кнопки рассылки также защищены проверкой на лету

---

## 🕐 Часовой пояс

Дни недели («сегодня» / «завтра») определяются по часовому поясу **Asia/Yekaterinburg (UTC+5)**. При необходимости можно изменить в `api.js`.
